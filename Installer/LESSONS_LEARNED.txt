For VirtualBox:
	Have to setup 2 adapters in each VM, one to access the internet, one to access
	other VMs and the host:
	Set the first adapter to NAT and second adapter to Host-only adapter.

	sudo nano /etc/network/interfaces.d/setup
		# The loopback network interface
		auto lo
		iface lo inet loopback
		# The primary network interface - Internet Access
		auto enp0s3
		iface enp0s3 inet dhcp

		# Alice
		# The network interface to Host and other VMs
		auto enp0s8
		iface enp0s8 inet static
